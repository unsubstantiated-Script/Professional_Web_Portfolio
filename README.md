# Project Title

Professional Portfolio 

## Description

Personal portfolio listing out some of the projects I've been working on. Has been built on EJS to get a scratch built mail server working and for SSR speed. 

### Key Dependencies

- EJS
- JQuery
- NodeMailer
- Request

### Installing

- npm i
- npm run kiffit
